# 📊 A Data Analysis of Sexual Violence in Bangladesh

<img width="1920" height="1080" alt="Sexual Violence Analytics in BD (1)" src="https://github.com/user-attachments/assets/802ca533-8194-4d08-b320-6ab639e6d84c" />

## 📌 Project Overview
This project focuses on analyzing reported cases of sexual violence in Bangladesh using data-driven techniques. The goal is to understand patterns based on time, location, victim type, season, and other important factors.

The analysis includes data preprocessing, handling missing values, transformation of numerical ranges, and multiple visualizations to extract meaningful insights.

---

## 📂 Dataset Information

The dataset contains the following features:

- id  
- ageband  
- victim_type  
- victim_offender_relation  
- incident_upazila  
- incident_district  
- incident_division  
- part_of_the_day  
- incident_date  
- year  
- month  
- day  
- temperature  
- season  
- sexual_violence_type  
- source  
- news_link  

---

## 🧹 Data Preprocessing

The following preprocessing steps were applied:

- Handled missing values  
- Converted age ranges into mid-point values  
- Standardized categorical variables  
- Cleaned inconsistent entries  
- Converted date-related fields into structured format
